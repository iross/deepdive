ocr_pdf
^^^^^^^

Driver of the Optical Character Recognition (OCR) process.

Usage::
=======

    python ocr_pdf.py --tesseract --no-cuneiform --pdf
    
Would run the tesseract OCR package and return the results embedded into a new PDF document.
